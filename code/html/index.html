Présentation de l'application

Cette application permet de suivre votre cycle menstruel avec l'aide de l'intelligence artificielle pour prévoir vos périodes fertiles, les symptômes et optimiser votre bien-être.

Nos Services

Voici quelques-uns des services que notre application vous propose :

  • Suivi personnalisé : Organisez vos journées en harmonie avec votre cycle menstruel.
  • Alertes de fertilité : Recevez des notifications pour connaître vos jours de fertilité maximale.
  • Analyse des symptômes : Suivez les effets hormonaux et anticipez les changements dans votre corps.
  • Conseils de santé : Recevez des recommandations pour mieux gérer votre cycle menstruel.
